Cloud Integration
-----------------

H2O is supported on a number of cloud environments, including

- EC2 Instances and S3 Storage (RedHat AMI, Amazon Linux AMI, and Ubuntu AMI)
- Microsoft Azure
- IBM DSX
- Nimbix Cloud
